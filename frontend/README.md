# pah-fm

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Run e2e tests
Install selenium-standalone
```make install_selenium_standalone```

### Lints and fixes files
```
npm run lint
```


## Additional tools
